# 🚗 Honda EK OBD2B 9141 Reader

A cross-platform Android application that transforms your smartphone into a professional diagnostic dashboard for Honda EK vehicles. Reads real-time ECU data using the ISO 9141-2 (OBD2B) protocol through a simple ELM327 WiFi adapter.

## Key Features
- 📊 **9+ Real-time Parameters** - RPM, speed, coolant temp, throttle position, and more
- 🎨 **Premium Custom Gauges** - Color-coded arcs with smooth animations
- ⛽ **Fuel Economy Tracking** - Live km/L calculations
- 📍 **Trip Computer** - Distance and fuel usage monitoring
- 📱 **Responsive Design** - Optimized for both portrait and landscape
- 🔌 **Wireless OBD2** - Connects via WiFi ELM327 adapter
- 🏎️ **Honda-Optimized** - Specifically tuned for Honda EK 9141 protocol

## Technical Stack
- **UI Framework**: Kivy 2.1.0
- **Language**: Python 3.12
- **Build Tool**: Buildozer 1.5.0
- **Target**: Android 5.0+ (API 21+)
- **Protocol**: ISO 9141-2 (OBD2B)
